# Code of Conduct

This repository follows a simple standard: be constructive, respectful, and clear.

## Our Standards

- Be kind and specific in feedback.
- Assume good intent, but be honest.
- Prefer clarity over performative criticism.
- Respect the fact that this repository reflects a personal brand and public profile.

## Unacceptable Behavior

- Harassment or personal attacks
- Bad-faith comments or dismissive behavior
- Spam, hostility, or intentionally disruptive contributions

## Enforcement

If something needs attention, contact the maintainer through the repository or via the contact links in the profile README.
