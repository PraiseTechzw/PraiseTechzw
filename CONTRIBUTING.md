# Contributing

Thanks for taking an interest in this profile repository.

## Scope

This repository is primarily for:

- the live GitHub profile README
- reusable profile README inspiration
- lightweight automation such as the contribution snake workflow
- documentation that helps others fork and adapt the structure

## Contribution Style

Please keep contributions:

- clear and professional
- easy to maintain on GitHub
- lightweight in dependencies and moving parts
- respectful of the owner's personal brand and links

## Before Opening a Change

- Prefer improving clarity over adding more widgets.
- Avoid adding sections that require frequent manual maintenance unless they provide strong value.
- Keep dynamic badges and external widgets intentional, not excessive.
- If you suggest template improvements, use placeholders instead of personal details.

## Suggested Areas

- copy improvements
- accessibility and readability
- template improvements
- documentation fixes
- GitHub Actions cleanup

## Pull Request Notes

- Describe the change briefly.
- Mention any GitHub rendering assumptions.
- Include screenshots only if the visual structure materially changes.
