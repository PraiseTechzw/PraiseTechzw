# Security Policy

## Supported Scope

This repository contains markdown, templates, static site assets, and GitHub workflow configuration.

## Reporting a Vulnerability

If you discover a security issue related to repository automation or published site behavior, please do not open a public issue first.

Report it privately to:

- `praisetechzw@gmail.com`

Include:

- a short description of the issue
- affected file or workflow
- steps to reproduce
- impact assessment if known

You will receive an acknowledgment as soon as reasonably possible.
